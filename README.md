# QR Redirect Website

This project is a simple web application that allows users to be redirected to specific pages when they scan a QR code. The website includes a homepage, an about page, and a contact page, along with the necessary styles and scripts to ensure a smooth user experience.

## Project Structure

```
qr-redirect-website
├── src
│   ├── index.html          # Main homepage of the website
│   ├── about.html          # Information about the project
│   ├── contact.html        # Contact form or information
│   ├── css
│   │   ├── style.css       # Main styles for the website
│   │   └── responsive.css   # Responsive design styles
│   ├── js
│   │   ├── main.js         # Main functionality
│   │   └── qr-redirect.js  # QR code scanning functionality
│   ├── images
│   │   └── .gitkeep        # Placeholder for images
│   └── assets
│       └── .gitkeep        # Placeholder for additional assets
├── qr-codes
│   └── .gitkeep            # Placeholder for QR codes
└── README.md               # Project documentation
```

## Features

- **Homepage**: Provides links to the about and contact pages.
- **About Page**: Contains information about the project.
- **Contact Page**: Includes a contact form for user inquiries.
- **QR Code Functionality**: Users can scan QR codes to be redirected to specific pages.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser to view the homepage.
3. Use the provided QR codes to test the redirection functionality.

## Usage

- Navigate through the website using the links provided on the homepage.
- Scan the QR codes to be redirected to the corresponding pages.
